# <a name="microsoft-graph-training-module---build-microsoft-teams-apps-with-the-microsoft-graph-sdk"></a>Microsoft Graph-Schulungsmodul – Erstellen Microsoft Teams Apps mit dem Microsoft Graph SDK

In diesem Modul erfahren Sie, wie Sie mit dem Microsoft Graph SDK für den Zugriff auf Daten in Microsoft 365 arbeiten, indem Sie eine Microsoft Teams-App mit ASP.NET Core erstellen.

## <a name="lab---build-microsoft-teams-apps-with-the-microsoft-graph-sdk"></a>Übung – Erstellen Microsoft Teams-Apps mit dem Microsoft Graph SDK

In dieser Übung erstellen Sie eine Konsolenanwendung mithilfe der Microsoft-Authentifizierungsbibliothek (MSAL), um auf Daten in Office 365 mithilfe der Microsoft Graph zuzugreifen.
